# Betable DevelopersThis is a repo for managing machine readable artifacts for the [Betable Developers APIs](https://developers.betable.com/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Betable Developers APIs](https://developers.betable.com/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://developers.betable.com/)- [Developer](https://developers.betable.com/)- [Twitter](https://twitter.com/betableapi)- [GitHub](https://github.com/betable)The [APIs.json](https://github.com/api-evangelist/betable-developers/blob/master/apis.json) index provides links to all the relevant links for the Betable Developers APIs, including the OpenAPI and Postman if they are present.
